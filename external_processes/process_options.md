# Process Options
## poEchoCmd
Echo the command to stdout before executing

## poUsePath
Search the PATH environment variable for the executable

## poEvalCommand
Evaluate the command using the system shell (e.g., cmd.exe or /bin/sh)

## poStdErrToStdOut
Merge error output (stderr) into the standard output stream

## poParentStreams
Share the parent's stdin/stdout/stderr handles
